# Crowd Sourced Civic Issue Reporting and Resolution System

## About the Project

This project is developed to help people report civic issues in their area. Users can submit complaints like potholes, garbage, water leakage, road damage, and street light problems. The admin can view all complaints, open the location using Google Maps, and mark the complaint as resolved.

## Technologies Used

* HTML
* CSS
* Python (Flask)
* SQLite
* Git & GitHub

## Features

* Home Page
* Report Complaint
* Admin Dashboard
* Resolve Complaint
* Google Maps Link
* SQLite Database

## How to Run

1. Download or clone the project.
2. Install Flask.
3. Run the following command:

```bash
python app.py
```

4. Open your browser and go to:

```
http://127.0.0.1:5000
```
